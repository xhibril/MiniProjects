## Tech Stack ▪️
- Java (JDK 24)
- JavaFX
- IntelliJ IDEA

## Features ▫️
- Track daily, monthly, and yearly spendings/earnings  
- Simple and responsive UI  
- Data saved locally  
- Visual summaries via `AreaChart`

## Run Locally ⚙️ 
**Requirements:** Java 24+, JavaFX.
1. Clone the repo
2. Run `Main.java`
3. Optional: load sample data from `sample_data/`
